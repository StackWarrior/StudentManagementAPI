# StudentManagementAPI
## Description
ASP.NET Core Web API project for managing students.

## Technologies
- ASP.NET Core 9
- Entity Framework Core
- Swagger 
- PostgreSQL
  
## Endpoints
- GET /api/students
- GET /api/students/{id}
- POST /api/students
- PUT /api/students/{id}
- DELETE /api/students/{id}
